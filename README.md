# ghostfrog-media

### Autonomous Content Factory & YouTube Feeder
`ghostfrog-media` is a specialized implementation of the **agentic-blueprint**. It uses **Documentation-Driven Orchestration (DDO)** to automate the research, production, and deployment of high-CPM YouTube Shorts.

## 🛡️ The YouTube Reliability Stack
* **AGENTS.md**: Roles for Research, Scripting, and the Ghost Driver.
* **CONSTRAINTS.md**: Hard Nos for AI-Slop, Copyright, and Quotas.
* **DECISION_LOG.md**: Memory of niche pivots and UI selector fixes.
* **SKILL.md**: Playbooks for "Ghost-Driving" the Gemini Web UI for $0 extra cost.
* **SCHEMA.md**: Postgres tables for video metrics and "Vibe-to-Profit" tracking.
* **RESEARCH.md**: The current active contract for the Search Agent.
